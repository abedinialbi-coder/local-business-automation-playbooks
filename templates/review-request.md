# Review Request Template

## Use case

A customer completes a successful appointment, service, consultation, or project.

## Basic workflow

1. Service is completed.
2. Thank-you message is sent.
3. Customer is asked for feedback.
4. Review link is sent when appropriate.
5. Negative feedback is routed privately before asking for a public review.

## Example message

Thanks for choosing us. If everything went well, would you be willing to leave a quick review?

## Best used by

- Local service businesses
- Clinics
- Agencies
- Home service companies
- Real estate teams
- Fitness studios

## Notes

Do not pressure customers. A review request should be respectful and should come after a positive service experience.
