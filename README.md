# Local Business Automation Playbooks

Practical CRM and follow-up automation playbooks for local businesses and agencies.

This repository collects simple workflow ideas, checklists, and message structures for local businesses that rely on calls, forms, appointments, quotes, consultations, and customer follow-up.

The playbooks are platform-neutral and can be adapted to different CRM and marketing automation systems, including GoHighLevel, HubSpot, Keap, ActiveCampaign, and similar tools.

## Included playbooks

- Missed call text-back workflow
- New lead follow-up workflow
- Appointment reminder workflow
- No-show recovery workflow
- Quote follow-up workflow
- Review request workflow
- Customer reactivation workflow
